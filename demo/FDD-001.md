# Feature: Place order from cart

**ID**: FDD-001
**Domain**: Order
**Actor**: Amara (Shopper)
**Status**: APPROVED

## Feature Statement

Place an order from the current cart so the shopper can complete their purchase.

## Business Value

Shoppers can checkout and receive an order confirmation, completing the purchase loop.

## Acceptance Criteria

- [ ] Given products exist, when the shopper opens the app, then they see a list of products with names and prices.
- [ ] Given the shopper has added items to their cart, when they view the cart, then they see the selected items, quantities, and total.
- [ ] Given the cart has at least one item, when the shopper places an order, then an order is created and they see a confirmation with an order ID.
- [ ] Given the cart is empty, when the shopper attempts to place an order, then the order is rejected with a clear message.

## Out of Scope

- Payment processing (separate feature)
- Order cancellation (separate feature)
- Stock level checks (separate feature)
