# Feature: Place order from cart

**ID**: FDD-001  
**Domain**: Order  
**Actor**: Amara (Shopper)  
**Status**: DRAFT  

## Feature Statement

Place an order from the current cart so the shopper can complete their purchase.

## Business Value

Shoppers can checkout and receive an order confirmation.

## Screens

| Screen ID | Screen Name        | Figma Status   |
|-----------|--------------------|-----------------|
| S01       | Product List       | ☐ Not started   |
| S02       | Cart               | ☐ Not started   |
| S03       | Order Confirmation | ☐ Not started   |

## Backend Endpoints

- GET /api/v1/products — list products
- GET /api/v1/cart — get current cart
- POST /api/v1/cart/items — add item (productId, quantity)
- POST /api/v1/orders — place order (from cart)

## Acceptance Criteria

- Given products exist, when the shopper opens the app, then they see a list of products.
- Given the shopper adds items to the cart, when they open the cart, then they see the selected items and quantities.
- Given the cart has items, when the shopper places an order, then an order is created and they see a confirmation with order ID.
