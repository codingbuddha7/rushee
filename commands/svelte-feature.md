---
name: svelte-feature
description: Implement a Feature Card (FDD) in the Svelte frontend. Same contract and pipeline as flutter-feature / react-feature.
usage: /rushee:svelte-feature <FDD-NNN>
example: /rushee:svelte-feature FDD-001
---

Invoke the **svelte-implementer** agent. Prerequisites: Feature Card, OpenAPI spec, screen inventory. Domain pure; no API in components. Output: feature with tests, ready for review.
